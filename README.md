# Natal da Família - Vue Project

Este projeto foi transformado de um arquivo HTML único para uma aplicação Vue 3 completa usando Vite.

## Estrutura do Projeto

- **src/components**: Componentes Vue modulares (Hero, Mensagem, Carrossel, etc.)
- **src/assets**: Estilos globais e recursos estáticos.
- **src/App.vue**: Componente principal que monta a página.

## Como Rodar

1.  Instale as dependências (se ainda não fez):
    ```bash
    npm install
    ```

2.  Rode o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```

3.  Acesse o link mostrado no terminal (geralmente `http://localhost:5173`).

## Funcionalidades

- **Carrossel Responsivo**: Implementado com Swiper.js.
- **Efeitos Visuais**: Neve (Canvas) e Luzes de Natal (SVG).
- **Áudio**: Player de música flutuante.
- **Design**: Estilo natalino moderno e responsivo.
